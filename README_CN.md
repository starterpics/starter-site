# ⚔️ 核心优势：Anti-Spam Architecture

### **从设计层面保证 Google SEO 安全**

与 Pinterest、壁纸站、内容农场不同，Starter.pics 的核心目标是：

> **零薄内容、零自动垃圾页、零 Google 风险。**

### 1) **隐藏标签系统（Tag ≠ 页面）**

标签永远不会自动生成 URL → 避免成千上万的薄内容页。

### 2) **人工精选页面系统（你创建的才是公开页面）**

只有你手动创建的页面才会公开：
→ 高质量
→ 完全可控
→ 绝不会误生成垃圾页

### 3) **AI Vision 长描述**

每张图片自动生成高质量描述：
→ 内容更丰富
→ 避免 “纯图片页面” 的 SEO 问题

### 4) **纯人工发布（MVP）**

没有自动化，没有 cron。
→ 当前版本采用 **最安全的 SEO 策略**

Starter.pics 的目标：
**成为全球 SEO 最安全的 AI 图片 Starter Kit。**

# ⚡ 技术护城河

### **高性能、低成本、可持续**

- **Prisma + PostgreSQL（Neon）**
  快速、可迁移、无供应商锁定。

- **浏览器端 WebP 预压缩 → 直传 Cloudflare R2**
  超低成本，避免服务器压力，无需 Vercel Image。

- **Next.js（App Router）**
  现代、稳定、未来可维护。

- **Tailwind + Shadcn UI**
  扩展性强的设计系统 + 极佳 DX。

- **Vercel AI SDK**
  内置 AI 描述能力，未来批处理工具基础。
